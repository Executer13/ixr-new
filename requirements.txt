# ============================================================================
# Platform-Specific Installation Notes
# ============================================================================
#
# WINDOWS:
# --------
# 1. Python 3.8+ (64-bit recommended)
# 2. Visual C++ Redistributable may be required for some packages
# 3. Bluetooth: Ensure BLE adapter is enabled in Device Manager
# 4. Firewall: Allow Python through firewall for LSL networking
#
# MACOS:
# ------
# 1. Python 3.8+ (via Homebrew recommended: brew install python@3.10)
# 2. Xcode Command Line Tools: xcode-select --install
# 3. Bluetooth: Grant Bluetooth permissions in System Preferences > Security & Privacy
# 4. May need to allow Python in Security & Privacy settings
#
# LINUX (Ubuntu/Debian):
# ----------------------
# 1. Python 3.8+: sudo apt-get install python3 python3-pip
# 2. Bluetooth: sudo apt-get install bluez
# 3. Bluetooth permissions: sudo usermod -a -G bluetooth $USER
# 4. Qt dependencies: sudo apt-get install python3-pyqt5 libxcb-xinerama0
# 5. OpenGL: sudo apt-get install mesa-utils libgl1-mesa-glx
# 6. Audio (optional): sudo apt-get install libasound2-dev
